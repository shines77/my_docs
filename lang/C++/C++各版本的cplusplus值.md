# C++ 各版本的 __cplusplus 值

## __cplusplus 值

在 C++98、C++11、C++14、C++17、C++20、C++23 标准中，`__cplusplus` 宏的预定义值分别为：

| 标准版本 | `__cplusplus` 值 |
|---------|------------------|
| C++98   | `199711L`        |
| C++11   | `201103L`        |
| C++14   | `201402L`        |
| C++17   | `201703L`        |
| C++20   | `202002L`        |
| C++23   | `202302L`        |

**补充说明：**

- 这些宏是由编译器根据当前启用的语言标准自动定义的。
- 某些编译器（如微软 VC++）在默认设置下可能不会严格报告这些值，需要添加 `/Zc:__cplusplus` 选项才能获得正确值。

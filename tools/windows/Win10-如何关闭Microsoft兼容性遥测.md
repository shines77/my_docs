
# Win10--如何关闭微软兼容性遥测

## 1. 起因

起初，微软兼容性遥测是微软为优化 Windows 系统性能和兼容性而设计的功能，其原理是通过收集用户设备的硬件和软件信息，帮助微软了解系统在不同环境下的运行情况，从而改进产品。

但正是由于它需要过收集用户设备的硬件和软件信息，来帮忙微软了解系统的原因。在系统使用的过程中，它会大量的占用 CPU 和硬盘的资源，从而造成明明系统没有运行什么软件，却发现系统老是卡顿，运行不流畅的情况出现。对于我们这些用户来说，实际用处不大。

其实，我们可以把这个【微软兼容性遥测】给关闭了，让它从此不在后台运行，不浪费占用我们的CPU和硬盘的资源，从而恢复电脑本应该有的运行速度。下面，我来和大家分享一下具体的关闭方法。

## 2. 关闭方法

首先，我们还是打开运行对话框，输入 [gpedit.msc]，打开组策略编辑器。先找到【计算机配置】，再下拉找到【管理模版】，再下拉找到【Windows组件】，在右边的窗口中，找到【数据收集和预览版本】。

找到【数据收集和预览版本】后，双击并打开它，在打开的窗口中，我们找到【允许遥测】或【允许诊断数据】，然后再双击打开它，把里面的设置改成【已禁用】，最后再点击应用确定就可以了。

最后再重新启动电脑就可以了。是不是很方便实用，快去试一试吧。

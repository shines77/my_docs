# Power Shell 分屏

## 1. 命令

- Shift + Alt + (-): 水平拆分窗口

- Shift + Alt + (+): 垂直拆分窗口

- Shift + Alt + 小键盘方向键: 调整分屏的大小，高度

- Shift + Ctrl + W: 关闭当前焦点所在的终端窗口

- Shift + Ctrl + D: 复制当前终端窗口

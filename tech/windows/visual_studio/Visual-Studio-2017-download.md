# Visual Studio 2017 下载

## Visual Studio 2017 完整版

Visual Studio 2017 版本 15.9 发行说明：[https://learn.microsoft.com/zh-cn/visualstudio/releases/2017/vs2017-relnotes](https://learn.microsoft.com/zh-cn/visualstudio/releases/2017/vs2017-relnotes)

最新版：[Visual Studio 2017 版本 15.9.78](https://learn.microsoft.com/zh-cn/visualstudio/releases/2017/vs2017-relnotes#15.9.78)

最新版默认会删除 `10.0.10240.0` 到 `10.0.17763.0` 的 `Windows SDK`，所以不太推荐。若要解决此问题，我们建议将项目重定向到 `10.0.19041.0`，或者根据需要将早期支持的版本重新定目标。

一个推荐的版本：[Visual Studio 2017 版本 15.9.24](https://learn.microsoft.com/zh-cn/visualstudio/releases/2017/vs2017-relnotes#15.9.24_SAR)

下载地址：
